# MCOS Bootstrap Build Spec v0.1

## Scope

MCOS/C-MORE v0.1 is a local-first Math Knowledge Graph bootstrap.

It is not the complete MCOS system.

## Required minimum

- MathObject;
- MathRelation;
- local validators;
- JSONL store;
- graph export;
- equivalence comparison;
- review packet;
- local ingestion;
- zero-distillation workflow;
- feed registry;
- selfdebug;
- tests.

## Not active in v0.1

- Lean adapter;
- Coq adapter;
- RDF backend;
- Neo4j backend;
- MCP server;
- external APIs;
- paid APIs;
- automatic promotion to core.

## Human gate

All promotions from staged material into core require explicit human review.
